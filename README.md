# paranormal-minds
source code for paranormal minds classroom system

# requirements
 * python2.7
 * node.js
 * google cloud sdk https://cloud.google.com/appengine/docs/standard/python/download
 
 
 # install dependencies

 `npm install -g gulp`

 `npm install`
 
 `pip install --upgrade google-api-python-client`
 

 
 # run project
 
 `npm start`
 
 # deploy

 `npm deploy`
